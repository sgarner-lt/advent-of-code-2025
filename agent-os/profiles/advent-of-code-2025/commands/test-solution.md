# Test AoC Solution

Run tests for an Advent of Code solution.

## Steps:

1. Identify the day/solution to test
2. Run tests with sample data first
3. Verify expected outputs match
4. Run with actual puzzle input
5. Report any failures or mismatches
6. If tests fail:
   - Show the difference between expected and actual
   - Help debug the issue
   - Suggest fixes
